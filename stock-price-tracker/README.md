# 📈 Stock Market Price Tracker

A beginner-friendly project to track real-time stock prices using **Python, Yahoo Finance API, Pandas, and Plotly**.

## 🚀 Features
- Fetches real-time stock data
- Calculates 20-day & 50-day moving averages
- Adds trend line analysis
- Interactive charts with Plotly

## ▶️ How to Run
```bash
pip install -r requirements.txt
python stock_tracker.py
